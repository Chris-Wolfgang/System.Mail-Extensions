Class MailMessageBuilder

Namespace
Wolfgang.Extensions.Mail

A fluent builder for constructing MailMessage instances. Simplifies the verbose ceremony of setting up mail messages.

public sealed class MailMessageBuilder

Examples

using var message = new MailMessageBuilder()
    .From("sender@example.com", "Sender Name")
    .To("alice@example.com", "bob@example.com")
    .Subject("Monthly Report")
    .HtmlBody("<h1>Report</h1>")
    .Attach("report.pdf")
    .Build();
